# The Data Campers - Data Science Bootcamp

This repository is designed to be a comprehensive resource for Data Science and Artificial Intelligence. It aims to include at least **2 example projects** for each of the following major topics:

*   Regression
*   Classification
*   Clustering
*   Computer Vision
*   Natural Language Processing (NLP)
*   Recommendation Systems
*   Time Series
*   Data Visualization
*   Deep Learning
